IDENTITY

Burstiness Index

A dimensionless measure quantifying temporal clustering patterns in discrete event sequences.

Diagnostic Temporal Distributional
Type: Descriptive Statistic
Position: Temporal structure characterization tool within the epistemic toolbox
Transformation
Maps inter-event interval distribution to a single scalar in [-1, 1]

EPISTEMIC ROLE

Function

Resolves the epistemic gap of temporal pattern quantification—transforming qualitative observations of clustering into precise, comparable measurements.

02
FORMAL CHARACTERIZATION

INPUT SPACE

Structure
T = {t₁, t₂, ..., tₙ}
Ordered timestamp sequence where tᵢ ∈ ℝ⁺
Derived Input
τᵢ = tᵢ₊₁ - tᵢ
Inter-event intervals (n-1 values)

TRANSFORMATION

B = σ - μ / σ + μ
Where σ = std(τ), μ = mean(τ)
σ = μ (Poisson) → B = 0
σ > μ (Bursty) → B > 0
σ < μ (Regular) → B < 0

OUTPUT SPACE

Artifact Type
B ∈ [-1, 1] ⊂ ℝ
Interpretation
-1 Regular | Random | Bursty +1

ASSUMPTIONS / PRECONDITIONS

Temporal Ordering

Events must be ordered chronologically; permutation destroys information.

Minimum Events

At least 3 events required (2 intervals) for variance computation.

Non-zero Intervals

Simultaneous events (τ=0) require special handling or exclusion.

CORRECTNESS CRITERIA

Theoretical Properties
Boundedness
B ∈ [-1, 1] for all non-negative intervals with σ > 0
Scale Invariance
B unchanged under uniform scaling of all intervals
Poisson Reference
B → 0 asymptotically for exponential interval distribution

TEST RESULTS

FAILURE MODES

Zero Variance

When all intervals are identical, σ = 0 and B is undefined (division by zero). Mathematically, lim σ→0 B = -1.

Small Sample Bias

With few events, sample variance underestimates population variance, biasing B toward -1. Minimum 30 intervals recommended.

Distribution Insensitivity

Different distributions can yield identical B values. Cannot distinguish between bimodal clusters and power-law tails.

COGNITIVE FRICTION

Common misunderstandings that systematically distort correct usage:

Confusion: B as Complete Descriptor

Treating B as fully characterizing temporal structure. Reality: B captures only one dimension (relative dispersion); many different processes can have identical B values.

Confusion: B = 0 Implies Poisson

Assuming B = 0 guarantees exponential intervals. Reality: Many non-exponential distributions can yield B ≈ 0. The converse holds asymptotically, not the inverse.

Confusion: Cross-domain Comparison

Comparing B values across fundamentally different domains without context. Reality: B = 0.5 in neural spikes has different significance than B = 0.5 in social media posts.

Confusion: Ignoring Sample Size

Treating B from small samples as reliable. Reality: Small samples produce biased, high-variance estimates. Always report sample size alongside B.

DIAGNOSTIC QUESTIONS
Q1
What happens if assumptions are violated?
If temporal ordering is lost (events shuffled), B becomes meaningless—it measures a property that no longer exists. If simultaneous events occur, they must be handled (excluded or jittered) to avoid zero intervals.
Q2
What invariant does this tool rely on most critically?
The relationship between mean and variance—specifically that the coefficient of variation (σ/μ) captures meaningful temporal structure. This assumes intervals are i.i.d. samples from some underlying distribution.
Q3
Can two different tools produce indistinguishable artifacts?
Yes. A bimodal distribution (many short intervals, few very long intervals) and a power-law distribution can both yield B ≈ 0.7, yet represent fundamentally different temporal processes. B alone cannot distinguish them.
Q4
How robust is B to outliers?
Not robust. A single extremely long interval dramatically increases σ while barely affecting μ, pushing B toward +1. Consider using robust alternatives (median-based) or trimming extreme intervals.
