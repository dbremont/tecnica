Epistemic Artifact Type

Concept

Conceptual Layer

Type Definition

A Concept is a structured epistemic artifact that partitions reality into instances and non-instances through a definable intension (conditions) and observable extension (members).

Epistemic Role

Concepts encode categorical affirmations about reality: they assert that certain things belong together by virtue of shared properties, enabling recognition, inference, and communication.

Instance Space

Intension

The set of defining properties that determine membership

Extension

The set of all entities satisfying the intension

Prototype

Canonical instance exhibiting all properties clearly

Boundary

Cases where membership is ambiguous or contested
