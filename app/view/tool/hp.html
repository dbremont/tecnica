Hawkes Process

A self-exciting point process for modeling temporal clustering. Each event increases future event probability, with influence decaying exponentially over time.

Identity

Generative Inferential
Name
Hawkes Process
Self-exciting point process
One-line Summary

Transforms temporal event sequences into probabilistic models capturing self-excitation dynamics, enabling prediction and triggering structure inference.

Formal Characterization

Input Space
Event Timeline: T = {"t₁, t₂, ..., tₙ"} ⊂ ℝ⁺
Observation Window: [0, T]
Output Space
Intensity Function: λ*(t) : ℝ⁺ → ℝ⁺
Parameters: μ, α, β
Transformation
λ*(t) = μ + Σᵢ α · e-β(t - tᵢ) · 𝟙(t > tᵢ)
Sum over all events tᵢ occurring before time t
Structural Assumptions
  • • Events are instantaneous points
  • • Excitation always positive (no inhibition)
  • • Exponential decay kernel
Stationarity Condition
  • • Branching ratio n = α/β < 1
  • • Finite expected intensity
  • • Time-invariant parameters

If the model is correct, transformed times Λ(tᵢ) should follow a unit Poisson process. The time transformation:

Diagnostic Questions

What happens if branching ratio exceeds 1?

Explosive regime: intensity grows without bound, infinite events in finite time. Estimation methods fail.

What invariant is most critical?

Exponential decay structure — encodes proportional influence decay. Violations require kernel modifications.

Can different parameters produce indistinguishable sequences?

Yes — identifiability issues when long-run mean intensity λ̄ = μ/(1-n) is similar across parameter sets.
